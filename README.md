# limehd-test-task

The project is a test task by Lime HD company

## Condition

1. Get to know api (https://swapi.co)
2. Request starships data.
3. Build a list of ships (use only names)
4. Implement list navigation.
5. When choosing a ship, you need to provide detailed information about it (at your discretion).
6. Detailed information includes a photograph of the starships.
   Example (https://starwars-visualguide.com/assets/img/starships/5.jpg)
   Where 5 in the file name is the ship id.
7. In the process of requesting the image should be displayed by the loader.
8. If the picture could not be obtained or not, then display the picture by default.
